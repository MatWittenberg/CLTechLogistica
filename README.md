# CLTech Logística 🚛

Este é o projeto final da disciplina de Linguagens de Programação. O **CLTech Logística** é um sistema de gerenciamento de transporte de cargas via console desenvolvido inteiramente em **Java**, focado na aplicação prática e avançada dos conceitos de Orientação a Objetos (POO).

## 🎯 Objetivo do Projeto
O sistema simula as principais operações de uma empresa de logística, automatizando o controle de mercadorias, rotas e caminhões. O grande diferencial do software é a sua inteligência de validação (Regra de Negócio), que impede o embarque de mercadorias caso a capacidade máxima de peso do caminhão seja excedida, evitando multas e prejuízos operacionais.

## 🛠️ Tecnologias e Arquitetura
- **Linguagem:** Java (JDK 8+)
- **Arquitetura:** MVC (Model-View-Controller) / Camadas (Serviços e Repositórios)
- **Paradigma:** Orientação a Objetos Avançada

## 🚀 Funcionalidades
1. Sistema de Login de Usuários com validação.
2. Cadastro de Produtos, Destinatários e Caminhões.
3. Simulação de Despacho (Venda) com cálculo automatizado de lotação de carga.
4. Geração de Relatórios dinâmicos com filtro inteligente por CPF.
5. Exclusão de Entregas e Cargas com recálculo automático de peso.
